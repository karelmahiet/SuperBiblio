Projet Spring SuperBiblio
C'st une application de gestion des bibliothèques créer avec le framework Spring.
Il inclut les fonctionnalités de Gestion des prêts et retours, des utilisateurs, reservations, commentaires etc.

- Pour accèder à l'application : http://localhost:8080
- Pour accèder à la Base de donnée : http://localhost:8080/h2-console
